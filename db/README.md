# Database

This folder owns the Prisma schema, migrations, and seed data for the Captar platform.

## Commands

- `pnpm db:generate`
- `pnpm db:push`
- `pnpm db:migrate`
- `pnpm db:seed`

## Seeded Demo Data

The seed creates:

- a demo user
- a demo project
- a live demo hook connection
- an active default hook policy
- a sample LLM session, trace, and spend ledger trail
